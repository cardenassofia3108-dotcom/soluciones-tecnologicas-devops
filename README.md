# Proyecto Final DevOps - Brenda Cardenas

Este proyecto demuestra la implementación de un flujo de trabajo CI/CD utilizando herramientas de AWS y GitHub Actions.

Contenido del Proyecto
* **Automatización**: Script de Python (`automatizacion.py`) utilizando la librería Boto3 para gestión de recursos.
* **Contenerización**: Dockerización de una aplicación Flask.
* **CI/CD**: Pipeline automatizado con GitHub Actions para integración continua.

 Tecnologías Utilizadas
* AWS Cloud9 & EC2
* Docker
* Python 3.9
* GitHub Actions
